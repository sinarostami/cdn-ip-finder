# CDN IP Finder for ShirOKhorshid
Tools to find and test CDN IPs that work in Iran for ShirOKhorshid CDN Fronting.

## Scripts
- `akamai_finder.sh` — Quick Akamai IP finder
- `akamai_iran_checker.sh` — Tests Akamai IPs from Iranian nodes
- `cdn_iran_checker.sh` — Tests ALL CDNs from Iranian nodes ⭐

## Usage
```bash
chmod +x cdn_iran_checker.sh
./cdn_iran_checker.sh
```
Paste output into ShirOKhorshid → Settings → CDN edge IPs
